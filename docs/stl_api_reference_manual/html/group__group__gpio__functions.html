uint8_t SelfTest_IO ( void  )

This function performs I/O tests to detect pin shorts to Ground or Vcc.

Not all pins maybe compatible with this test based on applications specifics. The users must fill up the "PinToTest" table corresponding to their needs.

Returns
0 - Test passed
1 - Test failed (Shorts to VCC)
2 - Test failed (Shorts to GND)
