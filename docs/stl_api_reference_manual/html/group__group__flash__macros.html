◆ LARGEST_NUM_OF_SUMS

#define LARGEST_NUM_OF_SUMS   (uint32_t)(359u)

The magic value 359 is the largest number of sums that can be performed without numeric overflow.

Given the possible initial starting value of sum1 = 0x1fffe. Any smaller value is also permissible; 256 may be convenient in many cases.
