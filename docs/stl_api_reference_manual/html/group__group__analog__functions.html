Parameters
baseThe pointer to a SAR ADC instance. For CAT1B, base is the group instance
channelChannel no. where the input voltage needs to be read
expected_resIf count_to_mV = 1 => Expected result in mV, else Expected result in counts
accuracyAccuracy in count ANALOG_ADC_ACURACCY
vbg_channelChannel no. where the VBG voltage is connected. Only for CAT1C device
count_to_mV1 = convert the count to mV.(take more time). Not applicable for CAT1B device
Returns
0 - Test passed
1 - Test failed
Note
For CA1B devices, User needs to configure the trigger input same as channel number
