◆ UartMesMaster_State()

uint8_t UartMesMaster_State ( void  )

Returns current master unit state.

Returns
UM_ERROR - last message process ended with error (the slave didn't respond or invalid responded data format) UM_COMPLETE - last message process succeeded (responded data is in the received buffer) UM_BUSY - unit is busy with message process
