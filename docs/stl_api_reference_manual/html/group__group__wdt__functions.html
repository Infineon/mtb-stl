uint8_t SelfTest_WDT ( void  )

This function perform Watchdog test.

Function runs WDT and wait defined period of time. If the PSoC was not reset function return ERROR status. If the PSoC was reset bit "1" in CyResetStatus must be set to "1". If this bit was set function return OK status.

Note
This function provides watchdog reset or stops software execution if watchdog fails
Returns
0 - Test passed 1 - Test failed
