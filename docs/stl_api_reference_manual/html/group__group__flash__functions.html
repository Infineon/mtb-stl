uint8_t SelfTest_FlashCheckSum ( uint32_t  DoubleWordsToTest)

This function checks for data corruption in flash memory using a checksum calculation.

Parameters
DoubleWordsToTestNumber of 32-bit Double Words of flash to be calculated per each function call.
Returns
1 - Test failed
2 - Test in progress
3 - Test completed

void SelfTest_Flash_init ( uint32_t  StartAddressOfFlash,
uint32_t  EndAddressOfFlash,
uint64_t  flash_ExpectedCheckSum 
)

This function checks the checksum of the flash for the given range of flash with the expected/reference checksum passed along with this API.

Parameters
StartAddressOfFlashStart address of the flash from where the checksum needs to be calculated.
EndAddressOfFlashEnd address of the flash till where the checksum needs to be calculated.
flash_ExpectedCheckSumExpected checksum. Must be stored outside the range of check.
Note
This function needs to be called prior to SelfTest_FlashCheckSum else the test will fail
