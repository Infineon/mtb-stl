void SelfTest_Motif_Init ( stl_motif_cfg_handle_t *  hPtr)

Initialize the Motif self test configuration.

Initialization includes

  • Generate the emulated signals for Phase-A, Phase-B and Index using TCPWM. These signals are input to the MOTIF module.
  • Configure the TCPWM counter to capture the Q-CLK resolution.
  • Configure the MOTIF module.
    Parameters
    hPtrPointer to the motif self test configuration handler.

uint8_t SelfTest_Motif_Start ( stl_motif_cfg_handle_t *  hPtr)

This function starts the Motif and capture the count between two quadrature clock edges.

Captured value compares with the reference value and returns the result.

Parameters
hPtrPointer to the motif self test configuration handler.
Returns
"0" "OK_STATUS" - Test passed
"1" "ERROR_STATUS" - Test failed
